Exercício - Multithreading

Para resolução destas questões, utilize as classes contidas neste arquivo

  1. Modifique a classe Pessoa para o que o código contido no método informarDesejo seja executado em uma thread separada do método main e de forma a permitir o paralelismo na "discussão" entre homem e mulher.
  2. Altere a classe Alarme para que fique constantemente verificando a hora e o minuto do sistema. A classe deve avisar as interfaces gráficas em formato de relógio digital e analógico quando a verificação for feita pela primeira vez ou quando houver alteração no valor da hora ou do minuto. A cada iteração, a thread deve esperar um tempo definido pela constante INTERVALO.
  3. Mude o comportamento da classe PortaRetrato para que repita o seguinte procedimento indefinidamente: obter a lista de arquivos de imagens contidas no diretório fornecido no construtor, para cada arquivo alterar a imagem do frame de imagem (método setImagem) e aguardar durante o intervalo de tempo passado como argumento no construtor.
  4. Uma thread deve salvar automaticamente, a cada minuto, o texto que estiver digitado. Este salvamento só pode ocorrer após a definição do arquivo que deve receber o texto através do atributo arquivo da classe Notepad (seja via opção salvar arquivo ou abrir arquivo). Dica: use o método salvarArquivo.
  5. Modifique o servidor HTTP (que pode ser acessado através de http://localhost:8080) para que, enquanto estiver executando, o tratamento de cada nova conexão estabelecida seja feita em um novo thread. O tratamento consiste em executar o método escreverSaida da classe TratadorDeRequisicoes.
